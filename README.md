# minimal-css-inject
Minimal CSS Injection Per Site

## How to Install

1. Save the files above into a single folder named css-injector.

1. Open Chrome and go to chrome://extensions.

1. Toggle Developer mode (top right corner).

1. Click Load unpacked.

1. Select your css-injector folder.

## Usage

1. Go to any website (e.g., example.com).

1. Click the extension icon.

1. Type body { background: #333 !important; color: white !important; }.

1. Click Save & Inject. The site should change immediately.

1. Refresh the page. The styles will persist.
